# 卡牌效率分析（超模/亏模）

基于标准费用模型，对376张卡牌进行效率分析，识别超模和亏模卡牌。

---

## 一、分析方法

### 1.1 价值计算公式

```
实际价值 = (基础数值 + 抽牌价值 + 关键词价值 + 状态价值) × AOE调整

效率比 = 实际价值 / 预期价值
```

### 1.2 参数设定

| 参数 | 数值 | 说明 |
|------|------|------|
| 基础伤害效率 | 7.0 伤/费 | 普通牌中位数 |
| 基础格挡效率 | 6.0 格/费 | 普通牌中位数 |
| 抽牌价值 | 4.9 伤害当量/抽 | 7.0 × 0.7折扣 |
| 1力量价值 | 4.5 伤害当量 | 持续收益 |
| 1敏捷价值 | 3.5 伤害当量 | 持续收益 |
| 1易伤/虚弱 | 1.0 伤害当量 | 状态效果 |
| 1Shiv/Soul | 3.5 伤害当量 | ≈0.5费价值 |
| 1Forge | 3.0 伤害当量 | 给君刃加伤害 |
| 1Summon | 1.5 伤害当量 | Osty召唤/治疗 |
| 保留/固有 | +3.5 伤害当量 | ≈+0.5费价值 |
| 虚无 | -3.5 伤害当量 | ≈-0.5费价值 |

### 1.3 稀有度调整

| 稀有度 | 预期溢价 |
|--------|----------|
| Common | 1.0x (基准) |
| Uncommon | 1.15x (+15%) |
| Rare | 1.3x (+30%) |

### 1.4 效率分级标准

| 分级 | 效率比范围 | 含义 |
|------|-----------|------|
| **严重亏模** | < 0.6 | 数值严重不足 |
| **轻度亏模** | 0.6 - 0.8 | 数值略低于预期 |
| **正常范围** | 0.8 - 1.2 | 符合标准模型 |
| **优秀卡牌** | 1.2 - 1.4 | 数值略高于预期 |
| **超模卡牌** | > 1.4 | 数值显著超标 |

### 2.1 效率分布

| 分级 | 数量 | 占比 |
|------|------|------|
| 严重亏模 (<0.6) | 204张 | 54.3% |
| 轻度亏模 (0.6-0.8) | 44张 | 11.7% |
| 正常范围 (0.8-1.2) | 61张 | 16.2% |

**观察**: 超过一半的卡牌被判定为"严重亏模"，这是因为模型仅计算了直接数值（伤害/格挡/抽牌/状态），而许多卡牌具有特殊的协同效果或场景价值，这些难以量化。

**亏模卡牌的共同特点**:
1. **特殊机制**: 许多卡牌具有模型难以量化的特殊效果（如击退、延迟、特殊召唤）
2. **协同依赖**: 需要特定卡组配合才能发挥价值（如All for One需要大量0费牌）
3. **场景价值**: 在特定场景下价值极高，但平均数值不高
4. **AOE惩罚**: 群体攻击因AOE调整而被低估
5. **长期收益低估**: Power牌和持续效果的回合收益难以精确计算

## 六、模型局限性说明

### 6.1 未计入的价值因素

1. **协同效果**: 如All for One与0费牌的配合
2. **场景价值**: 如击退在特定战斗中的关键作用
3. **长期价值**: Power牌和持续效果的回合收益
4. **卡组构建**: 某些卡牌在特定流派中的核心地位
5. **灵活性**: 可选择性效果的价值

### 6.2 使用建议

- **超模卡**: 通常可以直接加入卡组，数值优势明显
- **优秀卡**: 值得考虑，在合适的情况下表现良好
- **正常卡**: 标准选择，符合预期
- **亏模卡**: 需要特定配合或场景才能发挥价值，谨慎选择
