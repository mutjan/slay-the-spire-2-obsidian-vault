# 强力遗物选择攻略

## **S级遗物（改变游戏节奏）**

| 遗物 | 稀有度 | 效果 | 评价 |
|------|--------|------|------|
| **Glitter** | 远古 | 所有卡牌奖励带有 Glam 附魔 | 越早拿越强，每张新牌都升级 |
| **Snecko Eye** | 远古 | 每回合多抽2张，但获得混乱 | 高抽牌量，适合高费牌卡组 |
| **Runic Pyramid** | 远古 | 回合结束不再弃牌 | 保留关键牌，配合高费牌极强 |
| **Ice Cream** | 罕见 | 能量可以保留到下一回合 | 能量不浪费，每回合都能用满 |
| **Mummified Hand** | 罕见 | 打出能力牌后随机一张手牌本回合免费 | 能力流核心，大幅降低费用压力 |
| **Gambling Chip** | 罕见 | 战斗开始时可弃任意牌并抽等量牌 | 精准调度手牌，启动更快 |
| **Unceasing Top** | 罕见 | 手牌为空时抽一张牌 | 配合0费牌或小卡组可无限循环 |
| **Pandora's Box** | 远古 | 转化所有打击和防御牌 | 开局即删基础牌，大幅提升卡组质量 |
| **Empty Cage** | 远古 | 获得时移除2张牌 | 精准删牌，精简卡组 |
| **Black Star** | 远古 | 精英额外掉落遗物 | 遗物滚雪球，后期极强 |
| **Calling Bell** | 远古 | 获得时获得1个诅咒+3个遗物 | 高风险高回报 |
| **Astrolabe** | 远古 | 获得时转化3张牌并升级 | 精准改造卡组 |
| **Philosopher's Stone** | 远古 | 每回合+1能量，所有敌人+1力量 | 能量优势，适合速攻 |

## **遗物选择优先级建议**

1. **开局优先**：Pandora's Box、Empty Cage、Astrolabe（改造卡组）
2. **过牌优先**：Snecko Eye、Bag of Preparation、Pocketwatch
3. **能量优先**：Ice Cream、Philosopher's Stone
4. **成长优先**：Glitter、Black Star、Prayer Wheel、Kunai/Shuriken
5. **防御优先**：Tungsten Rod、Lizard Tail
6. **商店必买**：Chemical X（X牌流）、Orrery、Membership Card

---

**Glitter** 确实是后期最强遗物之一，它相当于让你每张新获得的牌都是升级过的，配合 Prayer Wheel 或 Black Star 这种增加奖励的遗物，卡组质量会滚雪球式增长。
