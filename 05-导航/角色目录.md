# 角色目录

```dataview
TABLE
  length(filter(file.inlinks, (f) => contains(f.path, "02-卡牌"))) as "卡牌数",
  length(filter(file.inlinks, (f) => contains(f.path, "03-遗物"))) as "遗物数",
  length(filter(file.inlinks, (f) => contains(f.path, "04-药水"))) as "药水数"
FROM "01-角色"
SORT file.name ASC
```
