# 故障机器人抓牌攻略

> **核心机制**: 充能球 (Orb) | 激发 (Evoke) | 频道 (Channel) | 等离子 (Plasma)

---

## 一、角色机制解析

### 1.1 充能球系统 (Orb)

故障机器人通过 Channel 生成充能球，每个球提供被动效果，Evoke 时触发强力效果。

**球类型**:
| 球类型 | 被动效果 | 激发效果 | 代表牌 |
|--------|----------|----------|--------|
| 闪电 (Lightning) | 回合结束对随机敌人造成伤害 | 对随机敌人造成大量伤害 | [[Ball Lightning]] |
| 冰霜 (Frost) | 回合结束获得格挡 | 获得大量格挡 | [[Cold Snap]] |
| 黑暗 (Dark) | 回合结束累积伤害 | 对目标造成累积伤害 | [[Darkness]] |
| 等离子 (Plasma) | 回合开始获得能量 | 获得能量 | [[Fusion]] |

### 1.2 频道与激发

**Channel (频道)**: 生成球进入球位
**Evoke (激发)**: 球被推出球位时触发

**频道手段**:
- [[Ball Lightning]] - 频道闪电
- [[Cold Snap]] - 频道冰霜
- [[Darkness]] - 频道黑暗
- [[Chaos]] - 随机频道

**激发手段**:
- [[Dualcast]] - 激发最右球两次
- [[Consume]] - 消耗球位获得属性
- [[Evoke]] - 主动激发

### 1.3 球位管理

球位决定可以同时存在的球数量。

**增加球位**:
- [[Capacitor]] - +2球位
- [[Inserter]] - 每回合+1球位

**消耗球位**:
- [[Consume]] - 消耗球位获得力量/敏捷
- [[Biased Cognition]] - 获得专注但球位-1

## 二、流派体系

### 2.1 闪电输出流 (推荐新手)

**核心思路**: 大量频道闪电球，通过被动和激发造成持续伤害。

**核心组件**:
```
闪电频道: Ball Lightning + Thunder Strike + Storm
闪电强化: Electrostatic Field + Static Discharge
辅助: Dualcast + Barrage
```

**关键卡牌**:
| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Ball Lightning]] | 1费 | ★★★★★ | 基础闪电频道 |
| [[Electrostatic Field]] | 1费 | ★★★★★ | 受击时频道闪电 |
| [[Thunder Strike]] | 3费 | ★★★★☆ | 闪电数量转伤害 |
| [[Storm]] | 1费 | ★★★★☆ | 能力牌频道闪电 |
| [[Dualcast]] | 1费 | ★★★★☆ | 双倍激发 |

**运作方式**:
1. 使用 Ball Lightning/Electrostatic Field 频道闪电
2. 回合结束闪电自动造成伤害
3. Dualcast 主动激发造成爆发
4. Thunder Strike 作为终结技

**优点**: 输出稳定，自动化高
**缺点**: 对单伤害分散

---

### 2.2 冰霜防御流

**核心思路**: 大量频道冰霜球，获得巨额格挡。

**核心组件**:
```
冰霜频道: Cold Snap + Glacier + Coolheaded
冰霜强化: Chill + Loop
防御配合: Barricade + Entrench
```

**关键卡牌**:
| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Cold Snap]] | 1费 | ★★★★★ | 攻击+冰霜频道 |
| [[Glacier]] | 2费 | ★★★★★ | 格挡+双冰霜 |
| [[Coolheaded]] | 1费 | ★★★★★ | 冰霜+抽牌 |
| [[Chill]] | 0费 | ★★★★☆ | 根据敌人数量频道冰霜 |
| [[Loop]] | 1费 | ★★★★☆ | 首球效果触发两次 |

**运作方式**:
1. 每回合频道多个冰霜球
2. 回合结束自动获得大量格挡
3. Loop 让冰霜效果翻倍
4. 几乎不需要主动防御牌

**优点**: 防御自动化，可以专注输出
**缺点**: 对多敌人分散

---

### 2.3 黑暗爆发流

**核心思路**: 累积黑暗球的伤害，一发激发秒杀。

**核心组件**:
```
黑暗频道: Darkness + Consuming Shadow + Doom and Gloom
黑暗加速: Amplify + Multi-Cast
爆发: Dualcast + Dark
```

**关键卡牌**:
| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Darkness]] | 1费 | ★★★★★ | 黑暗频道+触发 |
| [[Consuming Shadow]] | 2费 | ★★★★★ | 双黑暗+自动激发 |
| [[Amplify]] | 1费 | ★★★★☆ | 下张能力牌两次 |
| [[Multi-Cast]] | X费 | ★★★★☆ | 多次激发 |
| [[Rainbow]] | 2费 | ★★★★☆ | 三球各一 |

**运作方式**:
1. 使用 Darkness/Consuming Shadow 频道黑暗
2. 黑暗每回合自动累积伤害
3. 等待伤害累积到足够高
4. Multi-Cast/Dualcast 激发造成爆发

**优点**: 一发伤害极高
**缺点**: 需要准备时间

---

### 2.4 等离子能量流

**核心思路**: 频道等离子球获得大量能量，无限出牌。

**核心组件**:
```
等离子频道: Fusion + Meteor Strike
能量利用: Tempest + Thunder Strike
循环: Compile Driver + Skim
```

**关键卡牌**:
| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Fusion]] | 2费 | ★★★★★ | 频道等离子 |
| [[Meteor Strike]] | 5费 | ★★★★☆ | 高伤害+3等离子 |
| [[Tempest]] | X费 | ★★★★☆ | 能量转闪电 |
| [[Compile Driver]] | 1费 | ★★★★★ | 根据球类型抽牌 |
| [[Skim]] | 1费 | ★★★★☆ | 抽3张牌 |

1. Fusion/Meteor Strike 频道等离子

3. 使用多余能量打出更多牌

**优点**: 能量充足，出牌自由
**缺点**: 需要一定启动时间

---

### 2.5 专注叠加流

**核心思路**: 通过叠加专注(Focus)强化所有球的效果。

**核心组件**:
```
专注获取: Defragment + Biased Cognition + Core Surge
专注保持: Artifact + Orange Pellets
```

**关键卡牌**:
| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Defragment]] | 1费 | ★★★★★ | +1专注 |
| [[Biased Cognition]] | 1费 | ★★★★★ | +4专注但每回合-1 |
| [[Core Surge]] | 1费 | ★★★★☆ | 伤害+神器 |
| [[Consume]] | 2费 | ★★★★☆ | 消耗球位获得属性 |
| [[Capacitor]] | 1费 | ★★★★★ | +2球位 |

**运作方式**:
1. Defragment/Biased Cognition 叠加专注
2. Core Surge 提供神器抵消 Biased Cognition 的负面
3. 所有球的效果大幅提升
4. 少量球就能产生巨大效果

**优点**: 所有流派都能受益
**缺点**: 需要保护专注不被减少

## 四、战斗策略

### 4.1 开局策略 (Act 1)

- 优先拿取 Ball Lightning、Cold Snap
- 保持球位不空
- 适当拿取防御牌过渡

### 4.2 中期策略 (Act 2)

- 根据已有球类型确定流派
- 补充 Capacitor 增加球位
- 获取 Defragment 提升专注

### 4.3 后期策略 (Act 3-4)

- 确保有 Dualcast/Multi-Cast 爆发
- 保持至少一种防御手段
- 准备BOSS战长线输出

## 六、总结

故障机器人机制复杂但强大：

1. **闪电流** - 稳定输出，自动化高
2. **冰霜流** - 防御自动化
3. **黑暗流** - 一发爆发
4. **等离子流** - 能量充足
5. **专注流** - 全面提升

**新手推荐**: 闪电流，机制简单直观
**进阶推荐**: 专注流，全面提升所有球
