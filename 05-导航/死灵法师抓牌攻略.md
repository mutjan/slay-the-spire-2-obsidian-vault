 Cz# 死灵契约师抓牌攻略

> **核心机制**: 厄运 (Doom) | 灵魂 (Soul) | 召唤 (Summon) | 消耗 (Exhaust)

---

## 一、角色机制解析

### 1.1 厄运系统 (Doom)

死灵契约师的核心输出机制，敌人累积 Doom 值，达到一定阈值后触发即死效果。

**Doom 叠加**:
- [[Scourge|天灾]] - 基础 Doom 来源
- [[Blight Strike|枯萎打击]] - 伤害转 Doom
- [[Deathbringer|死亡使者]] - AOE Doom
- [[Countdown|倒计时]] - 每回合自动叠加

**Doom 爆发**:
- [[End of Days|末日]] - 大量 AOE Doom + 即死
- [[No Escape|无处可逃]] - 根据已有 Doom 追加
- [[Oblivion|遗忘]] - 每出牌叠加 Doom

### 1.2 灵魂系统 (Soul)

Soul 是特殊的 0 费消耗牌，可以大量生成用于触发效果。

**Soul 生成**:
- [[Capture Spirit|捕获灵魂]] - 敌人掉血生成 Soul
- [[Reave|掠夺]] - 伤害+生成 Soul
- [[Glimpse Beyond|窥视彼岸]] - 所有玩家获得 Soul
- [[Grave Warden|墓地守卫]] - 防御+生成 Soul

**Soul 收益**:
- [[Devour Life|吞噬生命]] - 使用 Soul 时召唤
- [[Haunt|萦绕]] - 使用 Soul 时敌人掉血
- [[Soul Storm|灵魂风暴]] - 根据 Soul 数量增伤

### 1.3 召唤系统 (Summon)

通过召唤生物协助战斗。

**召唤来源**:
- [[Devour Life|吞噬生命]] - 使用 Soul 时召唤
- [[Dirge|挽歌]] - X 费召唤 X 次
- [[Summon]] - 直接召唤

**召唤强化**:
- [[Necromastery|死灵精通]] - 召唤物强化
- [[Legion|军团]] - 召唤数量增加

## 三、流派体系

### 2.1 厄运即死流 (推荐新手)

**核心思路**: 快速叠加 Doom，触发即死效果秒杀敌人。

**核心组件**:
```
Doom 叠加: Scourge + Blight Strike + Countdown
Doom 加速: No Escape + Oblivion
即死触发: End of Days
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Scourge]] | 1费 | ★★★★★ | 基础 Doom + 抽牌 |
| [[Countdown]] | 1费 | ★★★★★ | 每回合自动叠 Doom |
| [[Blight Strike]] | 1费 | ★★★★☆ | 伤害转 Doom |
| [[End of Days]] | 3费 | ★★★★★ | AOE Doom + 即死 |
| [[No Escape]] | 1费 | ★★★★☆ | 追加 Doom |

**运作方式**:
1. Countdown 建立每回合自动叠 Doom
2. Scourge/Blight Strike 快速叠加
3. No Escape 根据已有 Doom 追加
4. End of Days 触发即死清场

**优点**: 无视血量，直接秒杀
**缺点**: 对 BOSS 需要大量 Doom

---

### 2.2 灵魂循环流

**核心思路**: 大量生成 Soul，通过使用 Soul 触发各种效果。

**核心组件**:
```
Soul 生成: Capture Spirit + Reave + Grave Warden
Soul 收益: Devour Life + Haunt + Soul Storm
辅助: Glimpse Beyond
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Capture Spirit]] | 1费 | ★★★★★ | 敌人掉血生成 Soul |
| [[Devour Life]] | 1费 | ★★★★★ | 使用 Soul 时召唤 |
| [[Haunt]] | 1费 | ★★★★☆ | 使用 Soul 时敌人掉血 |
| [[Reave]] | 1费 | ★★★★☆ | 伤害+生成 Soul |
| [[Soul Storm]] | 1费 | ★★★★☆ | Soul 数量转伤害 |

**运作方式**:
1. Capture Spirit/Reave 生成大量 Soul
2. Devour Life 让使用 Soul 时召唤
3. Haunt 让使用 Soul 时造成伤害
4. 一回合打出大量 Soul 触发连锁

**优点**: 资源循环，越战越强
**缺点**: 需要一定启动时间

---

### 2.3 召唤大军流

**核心思路**: 大量召唤生物，以数量压制。

**核心组件**:
```
召唤来源: Devour Life + Dirge + Summon
召唤强化: Necromastery + Legion
辅助: Bone Brew (药水)
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Devour Life]] | 1费 | ★★★★★ | 使用 Soul 时召唤 |
| [[Dirge]] | X费 | ★★★★★ | X 次召唤 |
| [[Necromastery]] | 1费 | ★★★★☆ | 召唤物强化 |
| [[Legion]] | 2费 | ★★★★☆ | 召唤数量+1 |
| [[Pot of Ghouls]] | - | ★★★★☆ | 专属药水，添加 Soul |

**运作方式**:
1. Devour Life 建立使用 Soul 召唤的机制
2. 大量生成 Soul 并使用
3. Dirge 直接召唤多个
4. Necromastery/Legion 强化召唤物

**优点**: 召唤物承担伤害和输出
**缺点**: 需要 Soul 支持

---

### 2.4 消耗复活流

**核心思路**: 利用消耗机制，从消耗堆复活强力卡牌。

**核心组件**:
```
消耗手段: Exhaust 牌 + 消耗效果
复活来源: Raise Dead + Resurrect
终结: 复活后的高费强力牌
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Raise Dead]] | 1费 | ★★★★★ | 从消耗堆拿回牌 |
| [[Resurrect]] | 2费 | ★★★★☆ | 复活并强化 |
| [[Bury]] | 4费 | ★★★★☆ | 高伤害，消耗 |
| [[Banshee's Cry]] | 6费 | ★★★★☆ | 超高伤害，消耗 |

**运作方式**:
1. 使用强力牌后让其进入消耗堆
2. Raise Dead 将其拿回
3. Resurrect 复活并强化
4. 重复使用强力终结技

**优点**: 重复使用最强卡牌
**缺点**: 需要消耗堆有牌

---

### 2.5 自残交易流

**核心思路**: 通过对自己施加 Doom 换取强力效果。

**核心组件**:
```
自残手段: Borrowed Time + Self-Doom 牌
收益转化: Reaper Form + Life Transfer
保护: Death's Door + Block
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Borrowed Time]] | 0费 | ★★★★★ | 对自己施加 Doom 换能量 |
| [[Reaper Form]] | 3费 | ★★★★★ | 攻击伤害转 Doom |
| [[Death's Door]] | 1费 | ★★★★☆ | 施加 Doom 时获得格挡 |
| [[Negative Pulse]] | 1费 | ★★★★☆ | 格挡+AOE Doom |

**运作方式**:
1. Borrowed Time 对自己施加 Doom 获得能量
2. Reaper Form 让攻击伤害也施加 Doom
3. Death's Door 将 Doom 转化为防御
4. 快速出牌形成爆发

**优点**: 节奏快，能量充足
**缺点**: 需要小心血量管理

## 六、战斗策略



### 6.1 开局策略 (Act 1)

- 优先拿取 Scourge、Blight Strike
- 适当拿取防御牌
- 遇到精英前确保有输出手段

### 6.2 中期策略 (Act 2)

- 根据已有核心牌确定流派
- 厄运流拿 End of Days、No Escape
- 灵魂流拿 Devour Life、Haunt
- 删除基础打击牌

### 6.3 后期策略 (Act 3-4)

- 确保有即死手段或高爆发
- 准备长线战斗的防御
- 卡组控制在 25-30 张

## 八、总结

死灵契约师机制独特，玩法多样：

1. **厄运流** - 即死机制，无视血量
2. **灵魂流** - 资源循环，越战越强
3. **召唤流** - 大军压境，数量取胜
4. **消耗流** - 复活强力牌
5. **自残流** - 高风险高回报

**新手推荐**: 厄运流，有 Countdown 就能玩
**进阶推荐**: 灵魂流，操作感强
**娱乐推荐**: 召唤流，大军压境
