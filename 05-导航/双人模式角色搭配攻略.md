# 双人模式角色搭配攻略

> **核心思路**: 角色互补 | 机制联动 | 资源循环 | 伤害放大

---

## 一、配合机制总览

### 1.1 团队配合卡牌类型

| 类型 | 代表卡牌 | 效果 |
|------|----------|------|
| **能量共享** | Energy Surge (Defect) | 所有玩家获得能量 |
| **格挡转移** | Demonic Shield (Ironclad) | 将自身格挡给队友 |
| **伤害放大** | Flanking (Silent) | 敌人受到队友双倍伤害 |
| **属性赋予** | Coordinate (Colorless) | 给队友力量 |
| **保护转移** | Intercept (Colorless) | 替队友承担攻击 |
| **连击加成** | Gang Up (Colorless) | 根据队友攻击次数增伤 |

### 1.2 角色定位分析

| 角色 | 定位 | 优势 | 辅助能力 |
|------|------|------|----------|
| **铁甲战士** | 坦克/输出 | 高防御、Exhaust循环 | 格挡转移、伤害承担 |
| **静默猎手** | 输出/控制 | 毒素、匕首、Debuff | 伤害放大、团队格挡 |
| **故障机器人** | 辅助/输出 | 能量生成、Orb系统 | 团队能量、Plasma频道 |
| **死灵法师** | 召唤/输出 | Doom即死、Soul循环 | 团队Soul、集体召唤 |
| **摄政王** | 输出/辅助 | 锻造成长、无色牌 | 给队友无色牌 |

## 二、强力搭配推荐

### 2.1 故障机器人 + 任何输出 (推荐新手)

**核心思路**: 故障机器人提供能量，队友疯狂输出

**配合机制**:
```
能量供给: Energy Surge (ALL players gain 能量)
         Ignition (给队友频道 Plasma)
输出爆发: 队友利用额外能量打出高费攻击
```

**最佳搭配**:
- **故障机器人 + 铁甲战士**: 能量 + Fiend Fire/高费攻击
- **故障机器人 + 摄政王**: 能量 + 锻造后的王权之剑

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Defect | Energy Surge | 团队能量 |
| Defect | Ignition | 给队友 Plasma |
| 队友 | 高费攻击牌 | 消耗额外能量 |

**优点**: 简单直接，能量充足
**缺点**: 故障机器人自身输出较低

---

### 2.2 铁甲战士 + 静默猎手 (经典攻防)

**核心思路**: 铁甲战士承担伤害并转移格挡，静默猎手专注输出

**配合机制**:
```
防御体系: Demonic Shield (Ironclad 格挡转移给队友)
         Tank (Ironclad 承担队友伤害)
         Sneaky (Silent 队友攻击时获得格挡)
输出放大: Flanking (Silent 让敌人受双倍伤害)
         Knockdown (Colorless 团队伤害放大)
```

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Ironclad | Demonic Shield | 0费转移格挡 |
| Ironclad | Tank | 保护队友 |
| Silent | Flanking | 2费双倍伤害 |
| Silent | Sneaky | 队友攻击时获得格挡 |

**运作方式**:
1. Ironclad 建立高格挡
2. Demonic Shield 将格挡转移给 Silent
3. Flanking 让敌人受到 Silent 双倍伤害
4. Silent 疯狂输出

**优点**: 攻守兼备，容错率高
**缺点**: 需要一定配合默契

---

### 2.3 死灵法师 + 摄政王 (资源循环)

**核心思路**: 死灵法师提供 Soul 资源，摄政王利用无色牌体系

**配合机制**:
```
资源供给: Glimpse Beyond (Necrobinder 给所有玩家 Soul)
         Legion of Bone (Necrobinder 集体召唤)
资源利用: 摄政王使用 Soul 触发各种效果
         Largesse (Regent 给队友无色牌)
```

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Necrobinder | Glimpse Beyond | 团队 Soul |
| Necrobinder | Legion of Bone | 集体召唤 |
| Regent | Largesse | 给队友无色牌 |
| Regent | Cosmic Concoction | 无色牌药水 |

**运作方式**:
1. Necrobinder 使用 Glimpse Beyond 给双方 Soul
2. Regent 利用 Soul 触发效果
3. Regent 给 Necrobinder 无色牌补充
4. 双方互相补充资源

**优点**: 资源循环，越战越强
**缺点**: 启动较慢

---

### 2.4 双输出爆发 (Silent + Regent)

**核心思路**: 两个高输出角色互相放大伤害

**配合机制**:
```
伤害放大: Flanking (Silent) + 摄政王高伤害攻击
连击加成: Gang Up (Colorless) 根据攻击次数增伤
Tag Team: 互相触发攻击特效
```

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Silent | Flanking | 队友双倍伤害 |
| Silent | Shadow Step | 下回合双倍伤害 |
| Regent | Conqueror | 王权之剑双倍 |
| Colorless | Gang Up | 连击增伤 |
| Colorless | Tag Team | 攻击连锁 |

**运作方式**:
1. Silent 使用 Flanking
2. Regent 使用 Conqueror
3. 双方同时打出高伤害攻击
4. Gang Up/Tag Team 进一步放大

**优点**: 爆发极高，快速击杀
**缺点**: 防御较弱，需要速战速决

---

### 2.5 双防御体系 (Ironclad + Defect)

**核心思路**: 双重防御保障，稳扎稳打

**配合机制**:
```
格挡体系: Ironclad 高格挡 + Demonic Shield 转移
         Defect 冰霜球自动格挡
         Rally (Colorless) 团队格挡
保护机制: Intercept (Colorless) 替队友承担攻击
         Tank (Ironclad) 伤害转移
```

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Ironclad | Barricade + Entrench | 格挡翻倍保留 |
| Ironclad | Demonic Shield | 格挡转移 |
| Defect | Glacier + Chill | 冰霜格挡 |
| Defect | Coolant | 冰霜强化 |
| Colorless | Rally | 团队格挡 |
| Colorless | Intercept | 保护队友 |

**运作方式**:
1. Defect 频道冰霜球自动获得格挡
2. Ironclad 使用 Entrench 翻倍格挡
3. Demonic Shield 转移格挡给 Defect
4. 双方都有巨额格挡

**优点**: 几乎无敌，容错率极高
**缺点**: 输出较慢，战斗时间长

---

### 2.6 召唤大军流 (Necrobinder + 任何)

**核心思路**: Necrobinder 提供集体召唤，队友专注其他

**配合机制**:
```
集体召唤: Legion of Bone (双方各召唤6)
Soul 共享: Glimpse Beyond (双方获得 Soul)
辅助配合: 队友提供能量/防御支持召唤物
```

**关键卡牌**:
| 角色 | 卡牌 | 作用 |
|------|------|------|
| Necrobinder | Legion of Bone | 团队召唤 |
| Necrobinder | Glimpse Beyond | 团队 Soul |
| Necrobinder | Devour Life | 使用 Soul 召唤 |
| 队友 | 辅助/防御牌 | 保护召唤流 |

**运作方式**:
1. Necrobinder 使用 Legion of Bone
2. 双方都有大量召唤物
3. Glimpse Beyond 补充 Soul
4. 召唤物承担伤害和输出

**优点**: 召唤物分担压力
**缺点**: 召唤流本身较弱，需要配合

## 四、搭配策略总结

### 新手推荐搭配

1. **故障机器人 + 铁甲战士**
   - 理由: 能量供给简单直接，铁甲战士容错率高
   - 策略: Defect 给能量，Ironclad 输出

2. **铁甲战士 + 静默猎手**
   - 理由: 攻守兼备，经典组合
   - 策略: Ironclad 防御，Silent 输出

### 进阶搭配

3. **死灵法师 + 摄政王**
   - 理由: 资源循环，后期强大
   - 策略: 互相补充 Soul 和无色牌

4. **静默猎手 + 摄政王**
   - 理由: 双输出爆发
   - 策略: 伤害放大叠加

### 娱乐搭配

5. **双防御 (Ironclad + Defect)**
   - 理由: 体验无敌的感觉
   - 策略: 堆格挡，慢慢磨

6. **召唤流 (Necrobinder + Defect)**
   - 理由: 大军压境
   - 策略: Defect 给能量，Necrobinder 召唤

## 五、实战技巧

### 5.1 沟通要点

- **能量回合**: Defect 使用 Energy Surge 前告知队友
- **爆发回合**: 使用 Flanking/Knockdown 前协调攻击
- **保护时机**: Intercept/Tank 用于保护残血队友

### 5.2 卡牌分配

- 团队牌 (Colorless) 优先给更适合的角色
- 防御牌给坦克角色
- 输出牌给输出角色

### 5.3 遗物选择

- 优先选择团队收益遗物
- 互补型遗物 (一个选攻击，一个选防御)
