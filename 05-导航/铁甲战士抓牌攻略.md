# 铁甲战士抓牌攻略

> **核心机制**: 消耗 (Exhaust) | 力量 (Strength) | 防御 (Block) | 脆弱 (Vulnerable)

---

## 一、角色机制解析

### 1.4 高效率防御牌

铁甲战士拥有游戏中最强大的防御体系，以下是各阶段最值得抓取的防御牌：

| 防御牌 | 费用 | 格挡值 | 优先级 | 说明 |
|--------|------|--------|--------|------|
| [[Shrug It Off\|耸肩无视]] | 1费 | 8格挡 | ★★★★★ | 防御+过牌，所有流派必拿 |
| [[Ghostly Armor\|幽灵铠甲]] | 1费 | 10格挡 | ★★★★★ | 高格挡 ethereal，前期神卡 |
| [[Feel No Pain\|无惧疼痛]] | 1费 | 3-4格挡/消耗 | ★★★★★ | 消耗流核心防御，被动叠甲 |
| [[Barricade\|壁垒]] | 2-3费 | ∞ | ★★★★★ | 格挡不消失，防御流核心 |
| [[Entrench\|巩固]] | 0-2费 | 翻倍 | ★★★★★ | 格挡翻倍，配合Barricade无敌 |
| [[Impervious\|金刚不坏]] | 2费 | 30格挡 | ★★★★☆ | 一次性巨额格挡 |
| [[True Grit\|坚毅]] | 1费 | 7格挡 | ★★★★☆ | 格挡+消耗，配合消耗流 |
| [[Flame Barrier\|火焰屏障]] | 2费 | 12格挡 | ★★★★☆ | 格挡+反伤，应对多段攻击 |

**防御牌选择建议**：
- **通用情况**：优先拿 Shrug It Off（过牌+防御）和 Ghostly Armor（高格挡）
- **消耗流**：Feel No Pain 提供被动防御，配合消耗牌每回合自动叠甲
- **防御流**：Barricade + Entrench 组合，格挡累积到数百
- **前期过渡**：Ghostly Armor 1费10格挡性价比极高

---

### 1.1 消耗系统 (Exhaust)

铁甲战士最核心的机制，通过消耗卡牌触发各种效果。

**消耗触发收益**:
- [[Dark Embrace|黑暗之拥]] - 消耗抽牌
- [[Feel No Pain|无惧疼痛]] - 消耗获得格挡
- [[Charon's Ashes|卡戎之灰]] - 消耗造成AOE伤害

**主动消耗手段**:
- [[Burning Pact|燃烧契约]] - 消耗抽2
- [[Fiend Fire|恶魔之火]] - 消耗手牌造成大量伤害
- [[Corruption|腐化]] - 技能牌变0费但自动消耗

### 1.2 力量体系 (Strength)

通过叠加力量提升所有攻击伤害。

**力量获取**:
- [[Inflame|燃烧]] - 直接+2力量
- [[Demon Form|恶魔形态]] - 每回合+2力量
- [[Limit Break|突破极限]] - 力量翻倍
- [[Spot Weakness|观察弱点]] - 敌人意图攻击时+3力量

### 1.3 防御体系

铁甲战士拥有游戏中最稳定的防御能力。

**核心防御牌**:
- [[Shrug It Off|耸肩无视]] - 格挡+抽牌
- [[Ghostly Armor|幽灵铠甲]] - 高格挡 ethereal
- [[Barricade|壁垒]] - 格挡不消失
- [[Entrench|巩固]] - 格挡翻倍

```
Act 1 开局
    │
    ├─ 拿到 Corruption ──→ 消耗循环流（最推荐）

    ├─ 拿到 Barricade ──→ 完美防御流

```

## 三、流派体系

### 2.1 消耗循环流 (推荐新手)

**核心思路**: 大量消耗卡牌，触发消耗收益形成资源循环。

**核心组件**:
```
消耗收益: Dark Embrace + Feel No Pain + Charon's Ashes
主动消耗: Burning Pact + Fiend Fire + Corruption
辅助过牌: Battle Trance + Offering
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Corruption\|腐化]] | 3费 | ★★★★★ | 核心，技能0费化 |
| [[Dark Embrace\|黑暗之拥]] | 2费 | ★★★★★ | 消耗抽牌引擎 |
| [[Feel No Pain\|无惧疼痛]] | 1费 | ★★★★★ | 消耗获得格挡 |
| [[Burning Pact\|燃烧契约]] | 1费 | ★★★★☆ | 消耗+过牌 |
| [[Fiend Fire\|恶魔之火]] | 2费 | ★★★★☆ | 终结技 |

**运作方式**:
1. 获取 Corruption 让技能牌0费化
2. 使用技能牌自动消耗触发收益
3. Dark Embrace 保证过牌不断
4. Feel No Pain 提供稳定防御

**优点**: 资源循环流畅，防御稳定
**缺点**: 需要 Corruption 启动

---

### 2.2 力量叠加流

**核心思路**: 快速叠加力量，一击必杀。

**核心组件**:
```
力量获取: Inflame + Demon Form + Spot Weakness
力量翻倍: Limit Break + Limit Break+
攻击载体: Heavy Blade + Sword Boomerang
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Inflame\|燃烧]] | 1费 | ★★★★★ | 快速启动 |
| [[Demon Form\|恶魔形态]] | 3费 | ★★★★☆ | 每回合成长 |
| [[Limit Break\|突破极限]] | 1费 | ★★★★★ | 力量翻倍 |
| [[Heavy Blade\|重刃]] | 2费 | ★★★★☆ | 力量收益最高的攻击 |
| [[Sword Boomerang\|飞剑回旋镖]] | 1费 | ★★★★☆ | 多次攻击触发力量 |

**运作方式**:
1. 前期使用 Inflame/Spot Weakness 叠加力量
2. Demon Form 提供每回合成长
3. Limit Break 将力量翻倍
4. Heavy Blade 造成巨额伤害

**优点**: 后期伤害爆炸
**缺点**: 启动较慢，前期较弱

---

### 2.3 完美防御流

**核心思路**: 建立不可突破的防御体系。

**核心组件**:
```
格挡累积: Shrug It Off + Ghostly Armor + Entrench
格挡保留: Barricade
格挡翻倍: Entrench + Entrench+
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Barricade\|壁垒]] | 3费 | ★★★★★ | 核心，格挡不消失 |
| [[Entrench\|巩固]] | 2费 | ★★★★★ | 格挡翻倍 |
| [[Shrug It Off\|耸肩无视]] | 1费 | ★★★★★ | 格挡+过牌 |
| [[Ghostly Armor\|幽灵铠甲]] | 1费 | ★★★★☆ | 高格挡 |
| [[Body Slam\|全身撞击]] | 1费 | ★★★★☆ | 格挡转伤害 |

**运作方式**:
1. 每回合使用防御牌累积格挡
2. Barricade 保证格挡不消失
3. Entrench 将格挡翻倍
4. Body Slam 将格挡转化为伤害

**优点**: 几乎无敌，容错率极高
**缺点**: 战斗时间长，需要耐心

---

### 2.4 自残输出流

**核心思路**: 利用掉血机制换取高输出。

**核心组件**:
```
掉血输出: Combust + Hemokinesis + Blood for Blood
掉血收益: Rupture + Brutality
回血: Reaper + Feed
```

**关键卡牌**:

| 卡牌 | 费用 | 优先级 | 说明 |
|------|------|--------|------|
| [[Combust\|自燃]] | 1费 | ★★★★★ | 掉血造成AOE |
| [[Rupture\|撕裂]] | 1费 | ★★★★★ | 掉血获得力量 |
| [[Hemokinesis\|御血术]] | 1费 | ★★★★☆ | 高伤害 |
| [[Reaper\|死亡收割]] | 2费 | ★★★★☆ | 吸血 |
| [[Blood for Blood\|以血还血]] | 4费 | ★★★☆☆ | 受伤后0费 |

**运作方式**:
1. Rupture 让掉血变成力量收益
2. Combust/Hemokinesis 主动掉血输出
3. Reaper 回血维持生存

**优点**: 伤害高，节奏快
**缺点**: 需要小心血量管理

## 六、战斗策略

### 6.1 开局策略 (Act 1)

- 优先拿取 Shrug It Off、Inflame 等基础牌
- 适当拿取攻击牌应对小怪
- 遇到精英前确保有防御手段

### 6.2 中期策略 (Act 2)

- 根据已有卡牌确定流派
- 删除基础打击牌
- 补充过牌和核心能力牌

### 6.3 后期策略 (Act 3-4)

- 精简卡组至25-30张
- 确保有过牌和防御
- 准备BOSS战专用牌

## 八、总结

铁甲战士是最适合新手的角色：

1. **消耗流** - 资源循环，稳定可靠
2. **力量流** - 后期爆发，一击必杀
3. **防御流** - 几乎无敌，容错率高

**新手推荐**: 消耗流，有 Corruption 就能玩
**进阶推荐**: 力量流，后期伤害爆炸
