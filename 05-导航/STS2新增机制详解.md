# Slay The Spire 2 新增机制详解

> 对比1代，2代带来了哪些全新体验？

---

## 概述

Slay The Spire 2 在保留1代核心玩法的基础上，为每个角色引入了全新的机制系统，同时增加了团队配合元素。本文详细介绍2代的新增内容。

---

## 一、全新角色机制

### 1.1 储君 (Regent) - 全新角色

**核心机制：聚焦 (Star) + 锻造 (Forge)**

储君是2代全新角色，拥有独特的资源系统和成长机制：

| 机制 | 1代 | 2代 (储君) |
|------|-----|-----------|
| 资源 | 无 | 聚焦 (特殊能量) |
| 成长 | 无 | 锻造系统 (永久强化) |
| 特色 | - | 王权之剑可不断升级 |

**新增关键词**：
- **聚焦 (Star)**：特殊资源，用于触发各种效果
- **锻造 (Forge)**：永久强化王权之剑的伤害
- **王权之剑 (Sovereign Blade)**：可升级的核心武器

**代表卡牌**：
- [[Alignment]] - 0费获得聚焦
- [[Furnace]] - 每回合自动锻造
- [[Refine Blade]] - 锻造+下回合获得聚焦

---

### 1.2 死灵契约师 (Necrobinder) - 全新角色

**核心机制：厄运 (Doom) + 灵魂 (Soul)**

死灵契约师是2代全新角色，专注于即死机制和召唤系统：

| 机制 | 1代 | 2代 (死灵契约师) |
|------|-----|----------------|
| 持续伤害 | 毒素 (Poison) | 厄运 (Doom) |
| 即死效果 | 无 | 有 (Doom达到阈值) |
| 召唤系统 | 无 | Soul + Summon |

**新增关键词**：
- **厄运 (Doom)**：累积型持续伤害，达到阈值触发即死
- **灵魂 (Soul)**：特殊的0费消耗牌
- **召唤 (Summon)**：召唤生物协助战斗

**代表卡牌**：
- [[Scourge]] - 施加Doom
- [[End of Days]] - AOE Doom + 即死
- [[Glimpse Beyond]] - 生成Soul

---

### 1.3 铁甲战士 - 机制强化

**新增机制：Star关联**

虽然铁甲战士的核心机制(Exhaust/Strength/Block)与1代相同，但2代增加了与"Star"相关的卡牌：

| 新增元素 | 说明 |
|---------|------|
| Star主题卡牌 | 如 [[Furnace]] (Star主题外观) |
| 团队配合 | [[Demonic Shield]] - 转移格挡给队友 |

**主要变化**：机制框架相同，卡牌池扩充，增加团队元素。

---

### 1.4 静默猎手 - 机制强化

**核心机制不变：毒素 + 匕首 + 弃牌**

静默猎手在2代保持了1代的核心玩法：

| 机制 | 1代 | 2代 |
|------|-----|-----|
| 毒素 | ✓ | ✓ |
| 匕首 | ✓ | ✓ |
| 弃牌 | ✓ | ✓ |
| 灵活 | ✓ | ✓ |

**新增元素**：
- 团队配合卡牌：[[Flanking]] - 让敌人受队友双倍伤害
- 新卡牌扩展战术选择

---

### 1.5 故障机器人 - 机制强化

**核心机制不变：充能球 (Orb) 系统**

故障机器人的Orb系统在2代保持不变：

| 球类型 | 1代 | 2代 |
|--------|-----|-----|
| 闪电 | ✓ | ✓ |
| 冰霜 | ✓ | ✓ |
| 黑暗 | ✓ | ✓ |
| 等离子 | ✓ | ✓ |

**新增元素**：
- 团队配合：[[Energy Surge]] - 全队获得能量
- [[Ignition]] - 帮队友频道Plasma

---

## 二、团队配合机制 (全新)

### 2.1 双人模式

**1代**：纯单人游戏
**2代**：支持双人合作

**新增配合元素**：

| 类型 | 代表卡牌 | 效果 |
|------|---------|------|
| 能量共享 | [[Energy Surge]] | 所有玩家获得能量 |
| 格挡转移 | [[Demonic Shield]] | 将格挡给队友 |
| 伤害放大 | [[Flanking]] | 敌人受队友双倍伤害 |
| 属性赋予 | [[Coordinate]] | 给队友力量 |
| 保护转移 | [[Intercept]] | 替队友承担攻击 |

### 2.2 无色团队牌

2代新增了一批专门用于团队配合的无色牌：

- [[Believe in You]] - 给队友能量
- [[Rally]] - 全队获得格挡
- [[Gang Up]] - 根据队友攻击增伤
- [[Tag Team]] - 攻击连锁

---

## 三、卡牌机制变化

### 3.1 新增卡牌类型

| 角色 | 1代卡牌数 | 2代卡牌数 | 新增 |
|------|----------|----------|------|
| 铁甲战士 | ~75 | 87 | ~12 |
| 静默猎手 | ~75 | 88 | ~13 |
| 故障机器人 | ~75 | 89 | ~14 |
| 死灵契约师 | - | 88 | 全新 |
| 储君 | - | 88 | 全新 |

### 3.2 新机制关键词

**储君专属**：
- Forge, Star, Alignment, Sovereign Blade

**死灵契约师专属**：
- Doom, Soul, Summon, Reaper Form

**通用新机制**：
- 团队配合关键词 (ally, other player)

## 四、游戏系统变化

### 4.1 敌人系统

**BOSS变化**：
- 部分1代BOSS回归 (The Guardian, Hexaghost, Slime Boss)
- 新增BOSS (待补充)

**敌人行为**：
- 意图系统保持不变
- 部分敌人新增团队模式下的配合

### 4.2 遗物系统

**新增遗物**：
- 支持新机制的遗物
- 团队配合相关遗物

### 4.3 药水系统

**角色专属药水**：
- 储君: [[Star Potion]], [[King's Courage]]
- 死灵契约师: [[Potion of Doom]], [[Bone Brew]], [[Pot of Ghouls]]

## 六、给1代玩家的建议

### 6.1 熟悉新角色

**优先尝试储君**：
- 机制直观：Forge永久强化
- 资源管理：聚焦的获取与消耗
- 成长感强：王权之剑越战越强

**死灵契约师进阶**：
- 理解Doom机制：累积即死
- Soul资源循环：生成与消耗
- 与传统Poison的区别

### 6.2 团队模式适应

- 关注队友角色和卡组
- 优先拿取团队配合牌
- 沟通关键回合的爆发

### 6.3 升级策略变化

**储君**：优先升级Forge相关牌
**死灵契约师**：优先升级Doom相关牌
**传统角色**：策略基本不变
