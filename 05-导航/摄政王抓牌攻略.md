### 1.1 核心资源：聚焦 (Star/Energy)

储君的独特资源是 **聚焦**（游戏内表现为特殊能量符号），通过特定卡牌获取并用于强化其他效果。

## 三、流派体系

### 2.1 聚焦循环流 (推荐新手)

**核心思路**: 大量获取聚焦，配合消耗聚焦触发效果的卡牌形成循环。

**核心组件**:
```
能量获取: Alignment + Glow + Solar Strike + Genesis
消耗触发: Child of the Stars (消耗聚焦获得格挡)
         Black Hole (消耗/获得聚焦造成AOE伤害)
         Orbit (每消耗4点聚焦获得1能量)
```

**关键卡牌**:
- [[Alignment|对齐]] ★★★★★ - 0费获取聚焦，必拿
- [[Child of the Stars|星辰之子]] ★★★★★ - 防御核心
- [[Black Hole|黑洞]] ★★★★☆ - AOE输出
- [[Genesis|创世纪]] ★★★★☆ - 每回合自动获取
- [[The Sealed Throne|封印王座]] ★★★★★ - 高牌数配合

**运作方式**:
1. 使用 Alignment/Glow 等获取聚焦
2. 消耗聚焦触发 Child of the Stars 获得大量格挡
3. Black Hole 提供AOE输出
4. Orbit 将消耗的聚焦转化为能量继续出牌

**优点**: 防御稳定，输出平滑，容错率高
**缺点**: 需要一定启动时间

---

### 2.2 锻造爆发流

**核心思路**: 快速积累 Forge 点数，通过强化后的王权之剑造成巨额伤害。

**核心组件**:
```
快速锻造: Refine Blade + Summon Forth + Furnace + The Smith
王权强化: Conqueror + Seeking Edge + Sword Sage
终结手段: 强化后的 Sovereign Blade
```

**关键卡牌**:
- [[Refine Blade|精炼之刃]] ★★★★★ - 锻造+下回合能量
- [[Furnace|熔炉]] ★★★★★ - 每回合自动锻造
- [[Summon Forth|召唤]] ★★★★★ - 将王权之剑置入手牌
- [[Conqueror|征服者]] ★★★★☆ - 双倍伤害
- [[Seeking Edge|寻刃]] ★★★★☆ - AOE化
- [[The Smith|铁匠]] ★★★★☆ - 一次性大量锻造

**运作方式**:
1. 前期拿取 Furnace 建立每回合锻造
2. 使用 Refine Blade/Summon Forth 快速积累 Forge
3. 使用 Conqueror 触发双倍伤害
4. 一发强化后的王权之剑清场

**优点**: 后期伤害爆炸，一击必杀快感
**缺点**: 前期较弱，需要保护手段过渡

---

### 2.3 0费快攻流

**核心思路**: 利用储君丰富的0费卡牌快速过牌、输出。

**核心组件**:
```
0费输出: Astral Pulse + Meteor Shower + Comet + Gamma Blast
0费功能: Alignment + Cloak of Stars + Particle Wall
过牌支撑: Glow + Big Bang + Decisions, Decisions
```

**关键卡牌**:
- [[Astral Pulse|星脉冲]] ★★★★★ - 0费14点AOE伤害
- [[Meteor Shower|流星雨]] ★★★★★ - 0费14点AOE+debuff
- [[Comet|彗星]] ★★★★★ - 0费33点伤害+debuff
- [[Gamma Blast|伽马爆裂]] ★★★★☆ - 0费13点伤害+debuff
- [[Decisions, Decisions|抉择]] ★★★★☆ - 0费抽3+额外效果
- [[Particle Wall|粒子墙]] ★★★★☆ - 0费9格挡，可循环

**运作方式**:
1. 尽可能多地拿取0费卡牌
2. 使用 Glow/Decisions, Decisions 过牌
3. 一回合打出大量0费卡牌
4. Pale Blue Dot/The Sealed Throne 作为收益点

**优点**: 节奏快，不卡费，过牌流畅
**缺点**: 需要一定卡牌数量支撑，怕诅咒牌

---

### 2.4 无色联动流

**核心思路**: 利用无色卡牌与 Arsenal、Spectrum Shift 等牌的配合。

**核心组件**:
```
无色生成: Spectrum Shift + Manifest Authority + Quasar + Bundle of Joy
无色收益: Arsenal (使用无色牌获得力量)
         Heirloom Hammer (复制无色牌)
```

**关键卡牌**:
- [[Arsenal|军械库]] ★★★★★ - 无色牌变成力量来源
- [[Spectrum Shift|光谱转移]] ★★★★☆ - 每回合生成无色牌
- [[Manifest Authority|权威显现]] ★★★★☆ - 防御+无色牌
- [[Bundle of Joy|欢乐束]] ★★★★☆ - 大量无色牌
- [[Heirloom Hammer|传家宝锤]] ★★★★☆ - 复制无色牌

**运作方式**:
1. 获取 Arsenal 作为核心
2. 使用各种方式生成无色牌
3. 每回合打出无色牌叠加力量
4. 力量提升所有攻击伤害

**优点**: 成长性强，后期无敌
**缺点**: 需要 Arsenal 才能启动

---

### 2.5 AOE清场流

**核心思路**: 利用丰富的AOE卡牌快速清理多敌人战斗。

**核心组件**:
```
AOE伤害: Astral Pulse + Meteor Shower + Dying Star + Seven Stars
         Crash Landing + Seeking Edge(强化王权之剑)
AOE减益: Crush Under + Resonance
```

**关键卡牌**:
- [[Astral Pulse|星脉冲]] ★★★★★ - 0费14点AOE
- [[Meteor Shower|流星雨]] ★★★★★ - 0费14点AOE+debuff
- [[Dying Star|垂死之星]] ★★★★☆ - 1费9点AOE-9力量
- [[Crash Landing|坠毁]] ★★★★☆ - 1费21点AOE
- [[Seven Stars|七星]] ★★★★☆ - 2费7×7点AOE
- [[Seeking Edge|寻刃]] ★★★★★ - 王权之剑AOE化

**运作方式**:
1. 多拿AOE卡牌
2. 面对多敌人时集中释放AOE
3. Dying Star/Resonance 削弱敌人输出
4. 快速清理杂兵后单挑BOSS

**优点**: 清图效率高，适合多敌人战斗
**缺点**: 对单伤害可能不足

## 六、战斗策略



### 6.1 开局策略 (Act 1)

**优先拿牌**:
1. 0费攻击牌 (Astral Pulse, Meteor Shower)
2. 聚焦获取牌 (Alignment, Glow)
3. 基础锻造牌 (Refine Blade)

**打法要点**:
- 利用0费牌快速清理小怪
- 适当拿取防御牌应对精英
- 建立基础锻造体系

### 6.2 中期策略 (Act 2)

**流派定型**:
- 根据拿到的核心牌确定流派
- 删除基础打击牌
- 补充过牌和防御

**关键选择**:
- 有 Furnace/多张锻造牌 → 走锻造流
- 有 Child of the Stars/Black Hole → 走聚焦流
- 有大量0费牌 → 走快攻流
- 有 Arsenal → 走无色流

### 6.3 后期策略 (Act 3-4)

**卡组优化**:
- 精简卡组，删除低效牌
- 补充稀有核心牌
- 确保有过牌和防御

**BOSS战要点**:
- 锻造流：提前几回合积累 Forge
- 聚焦流：保持聚焦循环，注意防御
- 快攻流：快速过牌，一回合爆发

## 八、总结

储君是一个以 **资源管理** 和 **成长积累** 为核心的角色：

1. **聚焦** 是通用资源，可用于防御、输出、能量循环
2. **锻造** 是长期投资，后期收益巨大
3. **0费牌** 极其丰富，快攻流可行性高
4. **AOE能力** 出色，清图效率高

**新手推荐**: 聚焦循环流，防御稳定，容错率高
**进阶推荐**: 锻造爆发流，后期伤害爆炸
**娱乐推荐**: 0费快攻流，操作感强，节奏快
