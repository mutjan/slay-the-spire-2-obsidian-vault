# Slay The Spire 2 知识库

[![Cards](https://img.shields.io/badge/Cards-504-blue)](./02-卡牌)
[![Relics](https://img.shields.io/badge/Relics-252-green)](./03-遗物)
[![Potions](https://img.shields.io/badge/Potions-63-purple)](./04-药水)

这是一个面向 Obsidian 的 Slay The Spire 2 完整数据知识库，包含所有卡牌、遗物、药水的详细信息，以及完整的抓牌攻略。

## 特点

- **完整数据**: 504 张卡牌、252 个遗物、63 瓶药水
- **5 个角色**: 铁甲战士、静默猎手、故障机器人、死灵契约师、储君
- **详细攻略**: 每个角色 4-5 个流派分析
- **双向链接**: 所有实体通过 WikiLink 相互连接
- **Dataview 支持**: 角色页面自动聚合相关数据
- **标签系统**: 支持按稀有度、类型、角色筛选

## 使用方法

1. 安装 [Obsidian](https://obsidian.md/)
2. 克隆此仓库或下载 ZIP
3. 在 Obsidian 中打开文件夹
4. 安装 Dataview 插件（已配置）
5. 打开 `05-导航/主页.md` 开始浏览

## 数据来源

卡牌数据来自 [cjodo/sts2](https://github.com/cjodo/sts2) 仓库。

## 技术说明

本知识库使用 [Obsidian Vault Creator Skill](./OBSIDIAN_SKILL.md) 自动生成，该 Skill 可用于创建任何结构化的 Obsidian 知识库。
