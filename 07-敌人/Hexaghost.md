# Hexaghost

> **类型**: BOSS | **章节**: Act 1 | **生命值**: 250-264

## 核心机制

**激活**: 根据你的当前生命值计算Inferno伤害

## 行动模式

| 行动 | 伤害 | 效果 |
|------|------|------|
| **Activate** | 0 | 根据你的HP计算下次Inferno伤害 |
| **Inferno** | 变量 | 多重攻击，向弃牌堆添加Burn |
| **Sear** | 6 | 攻击并向弃牌堆添加Burn |
| **Inflame** | 0 | 获得2力量 |

## 伤害计算

Inferno伤害 = 基于你当前生命值的百分比计算，生命值越低伤害越低

## 应对策略

- **治疗准备**: 在Inferno前尽量保持高血量
- **Burn管理**: Burn会污染牌库，注意清理
- **节奏控制**: 了解Activate后的Inferno是主要威胁

## 行动规律

通常模式: Activate → Inferno → Sear → Inflame → 循环

## 各角色应对

| 角色 | 策略 |
|------|------|
| 铁甲战士 | 高格挡抵挡Inferno |
| 静默猎手 | 快速输出，利用Flurry |
| 故障机器人 | 冰霜球自动防御 |
| 死灵契约师 | Doom无视防御持续伤害 |
| 储君 | 高爆发快速击杀 |
